\doxysubsection{Description détaillée}
Représente un produit simple du catalogue. 

Il s\textquotesingle{}agit d\textquotesingle{}un produit unitaire, non composite, dérivé de la classe abstraite {\ttfamily \doxylink{classdomain_1_1product_1_1_product}{Product}}. Ce type de produit ne contient ni sous-\/produits ni structure hiérarchique.

Cette classe est utilisée pour les cas standards de gestion de stock.

\doxysubsection{Description détaillée}
Classe de test manuelle de l\textquotesingle{}application de gestion de stock. 

Contient un ensemble de tests unitaires simples exécutés depuis la ligne de commande, sans utilisation de framework externe.

Cette classe couvre \+:
\begin{DoxyItemize}
\item La création de produits
\item L\textquotesingle{}ajout, la recherche et la suppression de produits
\item La génération automatique d\textquotesingle{}identifiants
\item La recherche par nom
\end{DoxyItemize}

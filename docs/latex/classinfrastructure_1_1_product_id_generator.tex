\doxysubsection{Description détaillée}
Générateur d\textquotesingle{}identifiants uniques pour les produits. 

Garantit l\textquotesingle{}unicité des IDs via un incrément interne.

Utilisé pour éviter la duplication d\textquotesingle{}identifiants produits lors de l\textquotesingle{}ajout de nouveaux éléments au stock.

Ce générateur est thread-\/safe via synchronisation explicite.

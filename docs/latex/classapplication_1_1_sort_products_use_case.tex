\doxysubsection{Description détaillée}
Cas d\textquotesingle{}usage \+: Trier les produits du stock par ordre croissant de prix. 

Ce cas d\textquotesingle{}utilisation propose deux approches \+: 
\begin{DoxyItemize}
\item Tri rapide via l\textquotesingle{}API Java standard (Comparator + sort) 
\item Tri manuel via l\textquotesingle{}algorithme de tri par sélection (à des fins pédagogiques ou de benchmarking) 
\end{DoxyItemize}Cette classe n\textquotesingle{}effectue aucun tri en place dans le dépôt \+: elle retourne une nouvelle liste triée.

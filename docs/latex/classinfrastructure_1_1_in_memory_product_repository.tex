\doxysubsection{Description détaillée}
Implémentation en mémoire de l\textquotesingle{}interface {\ttfamily \doxylink{interfaceapplication_1_1ports_1_1_product_repository}{Product\+Repository}}. 

Cette classe fournit une solution de persistance temporaire, idéale pour les phases de test, de prototypage ou d\textquotesingle{}exécution sans base de données.

Les données sont stockées dans une Map et sont perdues à l’arrêt de l’application.

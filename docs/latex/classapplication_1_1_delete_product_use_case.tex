Cas d\textquotesingle{}usage pour la suppression d’un produit du stock. 

Cette classe encapsule la logique métier liée à la suppression d’un produit existant. Elle effectue les validations nécessaires avant d’interagir avec le dépôt de persistance.

Ce use case appartient à la couche application et dépend du port {\ttfamily \doxylink{interfaceapplication_1_1ports_1_1_product_repository}{Product\+Repository}}.

\doxysubsection{Description détaillée}
Représente un produit abstrait appartenant à un catalogue. 

Cette classe sert de base pour les types de produits concrets (ex\+: \doxylink{classdomain_1_1product_1_1_simple_product}{Simple\+Product}). Elle encapsule les propriétés communes à tous les produits \+: identifiant, nom, prix, stock, ainsi que les informations de marque et de gamme.

Cette classe est abstraite et ne peut être instanciée directement.
